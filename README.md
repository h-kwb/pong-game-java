原初の対戦式テニスゲーム「Pong」を Java（Eclipse）で再現したシンプルなゲームです。  
プレイヤーは左側のパドルを操作し、右側のパドルは CPU が自動で動かします。

## 🎮 ゲーム概要

Java（Swing）で開発したテニス風 Pong ゲームです。  
状態管理（Game State）を用いた設計で、タイトル画面・難易度選択・プレイ画面・ポーズ画面を切り替える構造になっています。

---

## 🖥️ 開発環境

| 項目 | 内容 |
|------|------|
| 言語 | Java |
| GUI | Swing |
| 実行環境 | JDK 8 以上 |

---

## 🕹️ 操作方法

| キー | 動作 |
|------|------|
| W / S | パドル上下移動 |
| P | ポーズ |
| ↑ / ↓ | ポーズメニュー選択 |
| ENTER | 決定 |
| R | ゲーム終了後タイトルへ |

---

## 🧠 ゲーム状態管理

本ゲームは `gameState` によって画面や処理を制御しています。

| 状態 | 説明 |
|------|------|
| TITLE | タイトル画面 |
| DIFFICULTY | 難易度選択画面 |
| PLAY | ゲームプレイ中 |
| PAUSE | ポーズ画面 |

現在、ゲーム終了はフラグ管理を使用しています。  
将来的に `GAMEOVER` 状態へ統合予定です。
