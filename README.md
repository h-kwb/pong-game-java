# Pong Game (Java)

原初の対戦式テニスゲーム「Pong」を Java（Eclipse）で再現したシンプルなゲームです。  
プレイヤーは左側のパドルを操作し、右側のパドルは CPU が自動で動きます。

---

## 🎮 ゲーム概要

- 画面上を動くボールをパドルで打ち返すクラシックな Pong スタイルのゲーム
- プレイヤー：左側パドルを操作
- CPU：右側パドルを自動制御
- ボールが相手側の壁に到達すると得点
- シンプルなルールで気軽に遊べる内容

---

## 🛠 使用技術

- **Java**
- **Eclipse**（開発環境）
- **AWT / Swing** を用いた描画処理

---

## ▶ 実行方法

1. 本リポジトリをクローン  
  git clone https://github.com/h-kwb/pong-game-java.git
2. Eclipse にインポート  
3. Main.java を実行するとゲームが起動します
